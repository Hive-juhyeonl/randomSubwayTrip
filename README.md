# 🎲 랜덤 서브웨이 트립 (Random Subway Trip)

이 프로젝트는 서울 지하철역을 랜덤으로 추천하고,
주변의 볼거리/먹을거리를 제공하는 앱입니다.
아내와의 데이트 장소를 고민하던 경험에서 시작되었습니다.

## 1. 🎯 프로젝트 목적 및 기획 배경

* **목적:** 사용자에게 새로운 장소 발견의 즐거움을 제공합니다.
* **기획 배경:** 매번 데이트 장소를 정하는 어려움을 해결하고자... (이하 생략)

## 2. ✨ 핵심 기능 (v1.0)

* **랜덤 역 추천:** 서울 지하철 노선도 기반으로 역 추천
* **필터링:** 2호선, 4호선 등 특정 노선 필터링 기능
* **주변 정보:** 선택된 역 주변 볼거리, 먹을거리, 놀거리 정보 제공
* **커뮤니티:** 사용자 방문 후기 및 평점, 랭킹 시스템

## 3. 🛠️ 기술 스택 (예상)

* **플랫폼:** 하이브리드 앱 (Web, Android, iOS)
* **서버/DB:** Firebase (BaaS) - 초기 비용 최소화
* **데이터:** 서울시 공공데이터 (초기 데이터 구축용)

## 4. 🚀 향후 확장 계획

* 게미피케이션 (미션 시스템) 도입
* 수도권 및 전국 광역시 지하철로 서비스 확대
